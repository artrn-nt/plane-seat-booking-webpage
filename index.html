Description

Welcome on this webpage. The aim of this project was to design a responsive booking webpage / widget for an imaginary airway company by using web technologies : Vanilla JavaScript, HTML5, CSS3 and Sass preprocessor. The graphic interface enable the user to select a flight destination and some seats in the traveller cabin of a plane. Four types of seat are available. After picking flight destination and seats positions, the user can submit his order. Data of the current user selection are set in hidden inputs tags which can be submitted to a backend service. Dynamic login and register forms can also be reached using navigation buttons in the page header. Thank you for your interest.
